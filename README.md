# 🎬 Netlura – Clone da Netflix

Projeto desenvolvido durante a Imersão Front-End com IA da Alura.

🔗 Acesse o projeto: https://patapewagner.github.io/netlura/

## 📌 Sobre o projeto

O Netlura é uma interface inspirada na Netflix, desenvolvida a partir de uma imersão prática promovida pela Alura, com foco em construção de interfaces modernas e uso de IA no desenvolvimento.

Além do conteúdo base da imersão, realizei melhorias e ajustes próprios, principalmente em responsividade, animações e organização do layout.

## 🚀 Funcionalidades

- Menu responsivo com versão mobile (hamburger menu)
- Animação de entrada e saída do menu
- Layout adaptado para diferentes tamanhos de tela
- Cards interativos com efeito hover
- Navegação entre páginas (perfil e catálogo)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox, Media Queries, animações)
- JavaScript
- Font Awesome
- Google Fonts

## 🤖 Uso de IA no projeto

Durante o desenvolvimento, utilizei IA como ferramenta de apoio para:
- Estruturação e ajustes de layout
- Correção de bugs e refinamento de CSS
- Implementação de animações e interações

## 🎯 Aprendizados

- Criação de interfaces responsivas
- Manipulação de DOM com JavaScript
- Organização de código front-end
- Uso prático de IA no fluxo de desenvolvimento

## 🔗 Observação

Alguns links do projeto redirecionam para a plataforma da Alura, pois fazem parte da proposta original da imersão.
